avatar-iBatis-plugin
====================

根据输入的表结构，表名，以及目标package,自动在目标package下生成如下清单：

- #####Entity（POJO）

- #####DTO（实现序列化接口）

- #####Dao

- #####sqlmap.xml(包含对应resultType)

- #####dao.xml（spring注册dao）

菜单栏file下选择「avatar」
